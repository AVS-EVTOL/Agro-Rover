# 🌾 AgriRover – Smart Agriculture Automation Rover

An intelligent agriculture rover designed to **analyze soil conditions, optimize them, and automatically plant seeds** using real-time sensor data.

Built for an **Interschool Agricultural Innovation Competition**, this project secured **🥇 1st place for Innovation among 30 schools.**

---

# 🚜 Project Overview

AgriRover is a **smart farming robot** that automates the early stages of crop cultivation.

The rover performs:

1. **Soil condition analysis**
2. **Automatic soil improvement**
3. **Automated seed planting**

This reduces manual effort and helps farmers make **data-driven agricultural decisions**.

---

# 🔬 Key Features

### 🌱 Soil Analysis
The rover uses multiple sensors to understand soil conditions.

Sensors used:

- **DHT11** – Temperature & humidity
- **Soil Moisture Sensor**
- **pH Sensor**

Servo motors deploy the sensors **only when analysis is required**, reducing wear and improving efficiency.

---

### 💧 Smart Soil Optimization

Based on the **selected crop**, the rover automatically adjusts soil conditions:

- Adds **water** if soil moisture is low
- Adds **fertilizer** if pH is not optimal

This ensures the soil reaches the **ideal growth conditions** before planting.

---

### 🌾 Automated Seed Planting

Once soil conditions are optimized:

1. The rover activates the **seed planting servo**
2. Seeds are released into the soil
3. The planting cycle completes automatically

No manual intervention required.

---

### 📚 Crop Database

The rover uses a **pre-fed crop database** containing:

- Ideal soil moisture
- Optimal pH values

Example crops included:

- Wheat
- Rice
- Corn

This makes the system **practical and farmer-friendly**.
